An interactive digital birthday invitation — envelope → reveal → card.

---

## 📁 Project Structure

```
sheneli-birthday/
├── index.html          ← the invitation
├── images/
│   ├── baby.jpg        ← your baby photo (already added ✅)
│   └── glam.jpg        ← ⚠️ ADD YOUR GLAM PHOTO HERE (rename your file to glam.jpg)
└── README.md
```

---

## 📸 Adding Your Glam Photo

1. Take your glam photo from your phone
2. Rename it to **`glam.jpg`**
3. Drop it into the **`images/`** folder (replacing the placeholder)

---

## 🚀 Deploying to GitHub Pages (Free Link in 5 mins)

### Step 1 — Create a GitHub Account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a New Repository
1. Click the **+** icon → **New repository**
2. Name it: `sheneli-birthday`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload Your Files
1. Click **uploading an existing file** (shown on the new repo page)
2. Drag and drop **ALL** the files:
   - `index.html`
   - `images/baby.jpg`
   - `images/glam.jpg` ← make sure to add this!
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** tab of your repository
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch → **/ (root)** folder
5. Click **Save**

### Step 5 — Get Your Link 🎉
After ~1 minute, your invitation will be live at:
```
https://YOUR-USERNAME.github.io/sheneli-birthday
```

Copy that link and send it to your friends! 🥂
